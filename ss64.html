This password generator works exactly the same as the one on SS64.com.

The only differences are this one

  • Has more fields for services I commonly use
  • Renders more efficiently on mobile
  • Has some specific fixes already implemented for specific services which have stricter password requirements
  • Has your browser treat the main password field as though you are logging into a website rather than signing up for one, which makes autofill more effective and prevents it from asking you if you want to save a password every time
  • Has been converted from the original SS64 code into cached JS modules, so it loads faster.
